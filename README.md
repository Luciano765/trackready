# TrackReady

TrackReady is a frontend learning project focused on helping beginner and amateur motorcycle trackday riders prepare for and choose the right trackdays.

## Current Features

- Semantic HTML5
- Modern CSS
- Static comparison table
- Trackday guides

## Tech Stack

- HTML5
- CSS3
- Git
- GitHub

## Planned

- JavaScript
- React
- TypeScript
- Real trackday comparison
- User-created checklists
